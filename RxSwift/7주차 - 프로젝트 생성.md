Xcode 실행

- File - New - Project - App - MVVMRxSwift

터미널 실행

1. cd MVVMRxSwift
2. pod init
3. open podfile
4. pod 'RxSwift', pod 'RxCocoa' 입력
5. pod install --verbose
6. open .
7. MVVMRxSwift.xcworkspace 실행

Xcode 실행

1. info.plist 선택
2. storyboard 키워드 검색
3. Storyboard Name, Main storyboard file base name 두 항목 제거
